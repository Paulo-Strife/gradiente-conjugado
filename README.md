# Avisos

Cada run_benchmark é um arquivo .sh que criamos para rodar cada código do programa com as informações que precisamos, atente-se a rodar cada um na ordem certa para ver as diferenças entre um código e outro.

o run_test.ps1 é um arquivo para rodar no windows, caso tenha uma máquina windows ele funciona sem nenhum problema.

## makefile

O makefile consegue compilar todas as versões de código que temos no repositório, basta usar o comando make para compilar tudo, ou usar o comando make"versão que quer compilar" ex: make v2, para compilar cada versão individualmente.
